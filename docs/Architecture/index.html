Deployment Diagram

This is our deployment diagram of the planned architecture, using Wave for the communication between cars and LTE for the gateway car and the Web App. We also have a REST API that is connected to our DATABASE that we use to comunicate with the WebApp and the gateway cars

How does our Message Flow?

To illustrate the path of an emergency message: when a vehicle suffers an accident, it notifies the gateway vehicle, which photographs and films the accident site, and sends the information to the roadside unit. The roadsite unit relays the information to the webapp, processes the data, and notifies the emergency services about the accident, offering the users of the system the possibility to view detailed information about the accident.
